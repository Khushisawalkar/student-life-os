# 🎓 Student Life OS

A full-stack productivity system designed for students to manage academics, tasks, and daily life efficiently.

---

## 🚀 Overview

Student Life OS is a centralized platform that helps students:

* 📌 Manage daily tasks and deadlines
* 📊 Track productivity and performance
* 🧠 Organize academic workflow
* 💰 (Upcoming) Track expenses and budgets
* 📅 Plan schedules and routines

---

## 🏗️ Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* Axios
* Lucide Icons

### Backend

* Java Spring Boot
* Spring Data JPA
* Spring Security

### Database

* MySQL

---

## 📂 Project Structure

```
student-life-os/
├── frontend/     # React application
├── backend/      # Spring Boot API
├── database/     # SQL schema
```

---

## ⚙️ Setup Instructions

### 🔹 Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on: http://localhost:5173

---

### 🔹 Backend

```bash
cd backend
./mvnw spring-boot:run
```

Runs on: http://localhost:8080

---

### 🔹 Database

Create a MySQL database:

```sql
CREATE DATABASE studentos;
```

Update credentials in:

```
backend/src/main/resources/application.yml
```
